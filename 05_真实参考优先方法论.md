用途：定义猫图腾在“去广告感 / 去过度编排感”问题上的默认生成顺序与硬约束。
什么时候读：开始找参考、写作战卡、写 prompt、出关键帧前。
是不是唯一真源：是；关于“真实参考优先于纯文字 prompt”的方法论以本文件为准。

# 05 真实参考优先方法论

## 核心摘要
- 当前主流 AI 图像模型默认更容易生成“干净、完整、像商业图”的画面，而不是天然生成真实社交平台里的生活流抓拍感。
- 所以“删几个道具 / 改几个形容词”只能做局部修补，不能稳定解决广告感。
- 猫图腾的默认顺序不是“先写 prompt 再希望自然”，而是“先找真实高赞参考结构，再让 AI 在该结构上生成变体”。
- prompt 的职责是精细控制，不是凭空发明自然感。

## 什么叫真实参考结构
- 来自真实社交平台、已被用户验证过的高赞猫类内容。
- 可拆解出明确的构图、光线、物件关系、猫的姿态、镜头距离与节奏。
- 不是 AI 图，不是商业图库，不是纯概念草图。

## 真实参考的定义升级（v2.0）

本文档v1.0版本将"真实参考"定义为"找一条真实的高赞参考视频"。

按 `08_市场快照与预测性研究方法论.md` 的方法论升级——

**v2.0的"真实参考"定义**：

- 不是单条参考视频
- 而是**已成功KOL/KOC群体的交集分析+预测下一阶段方向**的产物
- 单条参考视频是这个研究产物的一个具体载体，不是研究产物本身

**实际操作时的关系**：

- 先按08做研究池+交集+预测 → 得到方向预测
- 再按预测方向找单条参考视频 → 作为视觉/结构的具体借鉴
- 两步必须都做——只做后者属于"找一条参考"，可能是当前已饱和方向的参考

**v1.0版本的硬约束仍然有效**——本升级只是把"参考"的内涵扩展，不改变"真实参考优先于纯文字prompt"的根本原则。

## 硬约束
1. 没有真实参考结构，不进入作战卡。
2. 没有真实参考结构，不进入纯文字 prompt 试错。
3. 执行者 AI 的默认职责是“参考真实结构后生成变体”，不是“凭空想象一个自然画面”。
4. 如果生成结果第一眼像广告图、样片、摆拍图，第一反应不是继续改 prompt，而是回到参考池重找结构。
5. 所有“自然感”“生活感”“像真实帖子”相关目标，默认优先通过参考结构解决，不优先通过修辞形容词解决。

## 默认工作流
1. 先锁定情绪方向。
2. 再找真实高赞猫类内容作为参考源头。
3. 拆解其视觉结构：构图、光线、物件关系、猫的姿态。
4. 在该结构基础上写作战卡与 prompt。
5. 让执行者 AI 先出场景帧，再决定是否进入视频生成。

## 失败时怎么排查
- 如果画面“结构不错但像广告 / 像样片”，先怀疑参考结构不对。
- 如果画面“猫很精致但不真实”，先怀疑模型在默认商业图分布里收敛。
- 只有在真实参考结构已经明确的前提下，才继续做 prompt 微调。
