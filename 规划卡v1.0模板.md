# 规划卡v2.0模板

## 核心摘要
- 规划卡是下游执行模板，不是上游战略文件。
- 填卡前必须先读 `07_人的载体型剧本思维框架.md`，再读 `06_宏观框架先行原则.md`。
- 本卡中的“情绪内核”必须从具体心理状态出发，不得从动作出发。
- 本卡中的评估顺序必须遵守：投射密度 > 画面精细度。
- 规划卡未填写完成，不进入参考搜集、作战卡、关键帧、视频生成与复盘。

## 使用顺序
1. 先读 `08_市场快照与预测性研究方法论.md`
2. 再读 `07_人的载体型剧本思维框架.md`
3. 再读 `06_宏观框架先行原则.md`
4. 完成市场快照与5步推演链（粗版本即可）
5. 基于市场快照预测的方向+赛道，明确要服务的具体心理状态
6. 按本模板填写 7 个维度（v2.0新增维度0）
7. 完成 Go / No-Go 判断后，再进入执行层

## 评估维度

### 一级判断｜投射密度
- 目标用户能否在0.5秒内认出“这是我的状态”：
- 用户第一反应更像“认同自己”还是“评价猫”：
- 投射若不成立，问题最可能出在哪一层：

### 二级判断｜画面精细度
- 画面是否足够不阻碍投射：
- 风格统一是否轻度加强了投射：
- 是否存在会把心理状态带偏的精细化错误：

### 冲突处理
- 若投射密度与画面精细度冲突：以投射密度为准
- 若一级不过关：回剧本层重做
- 若一级过关 + 二级一般：允许继续进入后期
- 若一级与二级都过关：进入执行层

## Go / No-Go

### Go 条件
- 维度0市场快照对齐通过
- 赛道明确
- 心理状态明确
- 参考方向明确
- 猫适配方式明确
- 三层视觉分层明确
- 音频与文字层明确
- 一级判断通过

### No-Go 条件
- 还在跳过维度0直接做维度1-6
- 维度0研究池规模不足10个
- 还在从动作出发，而不是从心理状态出发
- 参考只是在展示动作事件，没有投射价值
- 主要评估仍停留在“精不精”“像不像”，没有回到投射密度
- 三层视觉结构没有共同服务同一个心理状态
