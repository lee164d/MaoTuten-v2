什么时候读：写 prompt 前、看图时、评估通过与否时。

# 03 什么样的内容算成立

## 核心摘要
- 猫图腾内容成立，不看"画得多复杂"，先看"投射密度是否成立"。
- 一级判断高于一切：目标用户第一秒是否进入那个心理状态，并识别为"这状态是我"。
- 画面精细度是二级判断，辅助一级判断不跑偏，不能替代一级判断。
- 视觉叙事的最小结构是：前景张力 / 中景体态 / 后景节奏替换，三层共同服务同一心理状态。
- 广告感 / 过度编排感默认视为失败信号，先回到真实参考结构检查，不优先靠 prompt 修辞补救。
- 对AI来说，不能画什么和必须画什么同等重要。

## 一级判断：投射密度
- 核心问题：目标用户在第一秒，能否认出"这就是我的状态"？
- 不看"比上次好不好"，只看"投射是否成立"。
- 不看"猫可不可爱"，只看"用户是否把自己投射进去"。
- 如果一级不通过，后面的构图、风格、参数都没有意义。

## 二级判断：画面精细度
- 画面精细度只做一件事：不让视觉上的粗糙/错位打断一级判断的投射。
- 精细度提升不能以牺牲投射密度为代价。
- 当精细度提升导致"广告感 / 样片感 / 摆拍感"出现时，精细度视为负贡献，回退优先。

## 一级 vs 二级冲突时的处理
- 一级过关 + 二级一般：允许继续进入后期。
- 一级不过关 + 二级精美：不通过，回到剧本层重做，不通过精修挽救。
- 一级与二级冲突不可共存时：以一级（投射密度）为准。
- 一级判断与07文档定义的心理状态冲突时：以07为准，回07重新校准心理状态。

## 三层情绪结构
### 前景
- 必须出现"本应关注但被搁置"的张力物件。
- 前景功能是放大心理状态，不是装饰。

### 中景
- 猫的体态必须传达该心理状态的外化形态。
- 重点不是表情丰富，而是体态与心理状态对位。

### 后景
- 必须用自然、缓慢、属于自己的节奏，替换社会催促节奏。
- 后景承担的是该心理状态的时间感。

## 正向成立标准
- 第一秒投射成立
- 猫仍然是猫，不是拟人角色皮套
- 情绪方向清晰，不是杂糅的概念拼贴
- 视觉元素各有情绪功能，不是为了好看硬堆
- 风格保持极简、克制、可识别
- 画面移除任一层（前/中/后）后，投射明显受损

## 负向约束 / 禁止项
- 禁止猫穿全身人类服装
- 禁止做不像猫的动作
- 禁止把猫画成几乎认不出的拟人角色
- 禁止依赖微表情做情绪转折
- 禁止为了"精致感"牺牲投射命中
- 禁止堆过多独立小物件制造伪丰富
- 禁止低俗、吵、乱、土
- 禁止没有真实参考结构支撑就凭纯文字想象构图
- 禁止出现明显广告式开场 / 过度编排感画面

## 通过的最低条件
- 用户看完不会说"结构不错但像个未完成样稿（demo）"
- 如果画面第一眼像商业图 / 样片 / 摆拍图，直接不通过
- 目标用户不是只看到"猫可爱"，而是会感到"这状态是我"
- 画面移除任一关键层后，情绪明显受损
